# 🐾 GrosiDog — Каталог собак питомника

> Учебный проект на Vue 3 + Vuex 4 без сборщика(CDN).  
> Каталог породистых собак питомника GrosiDog с фильтрацией, карточками и модальными окнами.

---

## ✨ Возможности

- 📋 **Каталог 13 собак** четырёх пород (немецкая овчарка, корги, шпиц, джек-рассел)
- 🔍 **Фильтрация по породе** с мгновенным обновлением списка
- 🪟 **Модальное окно** с подробной информацией о собаке
- 📸 **Лёгкая замена фото** — достаточно поменять одну строку в `store.js`
- 🏷️ **Бейдж «Продаётся»** для доступных к продаже собак
- 📱 **Адаптивная вёрстка** — работает на мобильном и десктопе
- ✅ **Без сборщика** — открывается напрямую в браузере

## 🚀 Запуск

Сборщик не нужен - проект работает напрямую в браузере.

### Вариант 1 - Live Server (рекомендуется)

1. Установите расширение **Live Server** в VS Code
2. Откройте папку проекта в VS Code
3. Нажмите правой кнопкой на `index.html` → **Open with Live Server**

### Вариант 2 - открыть файл напрямую

Просто откройте `index.html` двойным кликом.  
> ⚠️ При таком способе свои фото из папки `images/` могут не загружаться из-за ограничений браузера — лучше используйте Live Server.
